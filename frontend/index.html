🚀 Quick Setup Guide
1

Install & Configure Backend

Clone the project, install dependencies, and configure your .env file.

cd salon-system/backend
npm install
cp .env.example .env
2

Set Up MongoDB Database

Create a free cluster on MongoDB Atlas (atlas.mongodb.com) and paste your connection string in MONGODB_URI.

npm run seed # Load sample data
3

Create WhatsApp Business App

Go to developers.facebook.com → Create App → Add WhatsApp product. Get your Phone Number ID and Token.

4

Expose Local Server (Dev Mode)

Install ngrok and expose port 3000 for webhook testing.

ngrok http 3000
5

Register Webhook in Meta Dashboard

Callback URL: https://YOUR-NGROK-URL/webhook
Verify Token: same as your WHATSAPP_VERIFY_TOKEN

6

Start Server & Test

Start your server and send "hi" to your WhatsApp Business number!

npm run dev
7

Deploy to Production

Deploy on Railway, Render, or DigitalOcean. Update webhook URL in Meta dashboard. Set all env vars on the platform.
